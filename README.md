# Solo Leveling Status Window

A Solo Leveling inspired productivity RPG.

## Features

- Hunter progression system
- Rank system (E → Shadow Monarch)
- Daily task management
- Weekly dungeon tracking
- Achievement system
- XP progression
- Activity heatmap
- Analytics dashboard
- Damage penalties for missed tasks

## Tech Stack

- Astro
- TypeScript
- Tailwind CSS

## Status

Currently in active development.
