# Solo Leveling Status Window

A Solo Leveling inspired productivity RPG.

## Features

- Hunter progression system
- Rank system (E → Shadow Monarch)
- Daily task management
- Weekly dungeon tracking
- Achievement system
- XP progression
- Activity heatmap
- Analytics dashboard
- Damage penalties for missed tasks

## Tech Stack

- Astro
- TypeScript
- Tailwind CSS

## Usage Instructions

### Dungeon Gates & Quest Editing
- **Gate: ACTIVE (Opened):** Raid Mode is active. You can toggle checkmarks for task completions (Monday–Sunday) and proceed with daily raiding. In this mode, quest objective configurations are locked to prevent changes mid-raid.
- **Gate: LOCKED (Closed):** Customization Mode. You can add new weekly recurring tasks, edit descriptions, adjust reordering/weights, and delete quests. Completion logging is frozen in this mode.

*Note: This interface is designed primarily for desktop screens, with responsive swipe grid containers and modal drawer action menus for mobile viewports.*
